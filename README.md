#### 关于`sails.js`

`sails.js`类似于ruby on rails, 是一个优秀的web后台开发框架。它基于著名的`express`，添加了很多功能模块，支持`REST`、各种流行数据库等，使用它开发适用于多设备、数据密集型、实时web应用是件异常简单、高效、愉快的事情。
